11. Install Workspace Matrix extension and configure number of workspaces required. Also update keyboard shortcuts to move between workspaces.
    Use dconf-editor if some customizations do not show up in the keyboard shortcuts configuration application.
    For example
    gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-up "['<Ctrl><Super>k']"
    gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-down "['<Ctrl><Super>j']"
